## 1. 项目简介
#### 该项目使用了以下技术：
* Spring Boot
* Mybatis Plus
* MySQL

## 2.开发要求
#### 1. 项目使用Spring Boot框架，使用Mybatis Plus作为ORM框架，使用MySQL作为数据库。
#### 2. 项目使用Maven作为构建工具，使用Git作为版本控制工具。
#### 3. 项目开发人员请使用IntelliJ IDEA作为开发工具。
#### 4. 项目开发人员建立自己的分支，开发完成后合并到主分支。分支命名规则：develop-backend-姓名拼音或者昵称。
#### 5. 项目JDK要求：JDK 20
